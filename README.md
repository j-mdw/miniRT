MiniRT - 42

Run:
make
./minirt FILE_NAME.rt

Save bmp format:
./minirt FILE_NAME.rt -save

Keybinds:

Movements:
Move: arrows
Look up: s
Look down: w
Look right: d
Look left: a

Other actions:
Render: r
Change camera: n
Select object: left click
Unselect object/select camera: right click
Select light: l
Increase selected object size: z
Decrease selected object size: x
